Logo Project: Whale Design Studio

This webpage was created to showcase the logo i created for whale design studio. it includes but is not limited to the process of creating the logo.

Here are some of my thoughts about creating this logo design:

  • I chose to use an analogous color scheme for this logo.. etc..
  • i feel this logo appeals to the target audience for several reasons. etc...
  • the name of the business, Whale Design Studio has a quirky sort of feel. the whale icon fits the quirky kind of vibe
